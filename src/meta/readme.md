## Meta components are for internal usage in this project, to simplify some things
### Quicknav [live stream where it was created](https://youtube.com/live/ZvHCXjkvluE?feature=share)
### Component that helps you navigate projects in a Tutorial-like fashion
This component let's you add current selected node to the list of "Navigation Points". Once added, you can jump between the nodes in the list and they will be focused in the current viewport. For correct functioning of this component it is assumed that you have your viewport be pane index 0 (should be opened first, before any other panes in the project)
